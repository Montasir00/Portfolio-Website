# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| 1.x.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it responsibly by emailing **fazlurrahaman365@gmail.com**.

We take all security reports seriously and will acknowledge your report within 48 hours. Please do not disclose the vulnerability publicly until we have had a chance to address it.
